# **App Name**: Chirimoyo 33

## Core Features:

- Property Listing: Showcase the vacation home with high-quality photos and detailed descriptions of its features and amenities. Use the images in the /images folder.
- Availability Calendar: An interactive calendar to display real-time availability and booking status.
- Booking Form: A simple booking form for users to submit their reservation requests.
- Contact Information: Provide multiple contact options for inquiries, including email and phone number.
- Create ability to login via google. Create two user roles: Admin and Customer.

## Style Guidelines:

- Primary color: Sunny Yellow (#FFC107), reminiscent of the Spanish sun.
- Background color: Soft Sand (#F5F5DC), to create a warm and inviting atmosphere.
- Accent color: Terra Cotta (#E07A5F) to add a touch of Spanish culture.
- Body and headline font: 'Alegreya', a humanist serif, provides an elegant and intellectual feel for headlines and body text
- Use simple, modern icons representing amenities and local attractions.
- Responsive and clean design optimized for desktop and mobile devices, ensuring a seamless user experience. Angular and scss.
- Subtle animations to enhance user interactions, such as image transitions and form submissions.

## Technical spec:

- Use Angular as frontend
- Use Firebase for storage and hosting
